A simple and interactive Linear Search application built using **HTML, CSS, and JavaScript**. The program allows users to enter an array of numbers and a target value, then performs a sequential search to locate the element. The interface features a modern glassmorphism design with colorful gradients and responsive elements.

### Features
- Enter custom arrays and search values
- Sequential element searching
- Displays the index of the found element
- Responsive and user-friendly interface
- Stylish gradient and glassmorphism UI
